Project Overview

The Medical Calculators Collection is a comprehensive web-based tool that brings together over 250 clinical decision support calculators from UpToDate, organized by medical specialty. This project aims to provide healthcare professionals with quick, accessible, and reliable calculation tools for clinical practice.

The project is built entirely using HTML5, CSS3, and Bootstrap 5, ensuring a responsive, modern interface that works seamlessly across all devices.

Project Goals

  • Provide easy access to 250+ verified medical calculators
  • Support clinical decision-making across all major specialties
  • Offer both Clinical Criteria scoring systems and Medical Equations
  • Ensure responsive design for desktop, tablet, and mobile use
  • Maintain accurate calculations with proper unit conversions
Features

Core Features

  • Interactive dashboard with real-time progress tracking
  • Comprehensive calculator catalog with advanced search and filtering
  • Calculator organization by medical specialty (26 specialties)
  • Two calculator types: Clinical Criteria and Medical Equations
  • Build progress tracker showing completion status
  • Responsive Bootstrap 5 design
  • Unit conversion support (SI and conventional units)
  • Clean, intuitive user interface

Calculator Types

1. Clinical Criteria Scoring Systems

Purpose: Risk stratification and prognostic assessment using clinical parameters

Examples:

  • APACHE II Scoring System
  • CHA2DS2-VASc Stroke Risk Score
  • TIMI Score for MI
  • HAS-BLED Bleeding Risk Score

Output: Risk scores, mortality predictions, stratification categories

2. Medical Equations

Purpose: Calculate physiological, biochemical, and pharmacological parameters

Examples:

  • Body Mass Index (BMI)
  • Body Surface Area (BSA)
  • Cardiac Output
  • Glomerular Filtration Rate (GFR)
  • Unit Conversions (SI to Conventional)

Output: Numerical values with appropriate units and interpretation ranges

Usage Guide

Getting Started

  1. Open index.html in your web browser
  2. Browse the dashboard for quick statistics and overview
  3. Use the navigation menu to access different sections

Dashboard

View overall project statistics including:

  • Total number of calculators
  • Number of medical specialties
  • Build progress percentage
  • Completion status by specialty

Searching for a Calculator

  1. Go to the Catalog page
  2. Use the search box to find by name
  3. Filter by Clinical Criteria or Medical Equations
  4. View the description and category

Tracking Build Progress

  1. Navigate to Build Progress page
  2. View overall completion percentage
  3. Filter by status (Pending, In Progress, Completed)
  4. Track progress by specialty
Development Guide

Project Structure

calculators/ ├── index.html # Main dashboard ├── catalog.html # Calculator catalog ├── todo.html # Build progress tracker ├── docs.html # Documentation ├── data/ │ └── calculators.json # Calculator database └── calculators/ └── [individual calculator files]

Building a New Calculator

Each calculator should:

  • Be a standalone HTML file with embedded CSS and JavaScript
  • Use Bootstrap 5 for responsive design
  • Include input validation
  • Display clear results with interpretation
  • Support unit conversions where applicable
  • Include references to source calculations

Calculator Template

<!DOCTYPE html> <html> <head> <title>Calculator Name</title> <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet"> </head> <body> <div class="container"> <!-- Form inputs --> <!-- Calculate button --> <!-- Results display --> </div> <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script> <script> // Calculation logic </script> </body> </html>
Data Structure

Calculator Object Structure

{ "id": "unique-identifier", "name": "Calculator Display Name", "category": "Medical Specialty", "type": "Clinical Criteria|Medical Equation", "description": "What the calculator does", "inputs": [ { "name": "Input Field Name", "unit": "Unit of Measurement", "type": "number|text|boolean" } ], "formula": "Mathematical formula used", "status": "pending|in-progress|completed" }

Input Types

  • number - Numerical input with unit conversion
  • text - Text selection or dropdown
  • boolean - Yes/No or checkbox
Contributing

Contributing Guidelines

Contribution Process: Follow the development workflow to add new calculators or improve existing ones.
  1. Create Calculator File - Create a new HTML file for the calculator
  2. Update Database - Add entry to calculators.json
  3. Test Functionality - Verify all inputs and calculations
  4. Add Documentation - Include references and usage instructions
  5. Update Progress - Mark as completed in todo.html
Best Practices:
  • Use semantic HTML for accessibility
  • Implement responsive design
  • Validate all mathematical formulas
  • Include clinical references
  • Test across different screen sizes
  • Provide clear unit conversion options
