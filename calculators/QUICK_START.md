# Hospital Medicine Calculators - Quick Start Guide

## Get Started in 30 Seconds

### 1. Open the Master Index
Navigate to: `/home/user/clirnet/calculators/calculators/hospital-medicine-index.html`

Open this file in any web browser (Chrome, Firefox, Safari, Edge)

### 2. Search or Browse
- Use the search bar to find a specific calculator
- OR browse by category (Scoring Systems vs. Clinical Calculations)

### 3. Click "Calculate"
Select any calculator card to open it directly

### 4. Enter Patient Data
Fill in the required fields with patient values

### 5. View Results
Click the calculate button to see:
- Primary calculation result
- Risk classification (Low/Moderate/High)
- Clinical interpretation
- Treatment recommendations

## Usage Tips

### Search Effectively
- Search by calculator name: "APACHE", "Wells", "Braden"
- Search by condition: "pneumonia", "sepsis", "burn"
- Search by calculation: "mortality", "risk", "severity"

### Understanding Results
- **Green indicators** = Low risk, good prognosis
- **Yellow indicators** = Moderate risk, caution advised
- **Red indicators** = High risk, aggressive management needed

### Mobile Usage
- Full functionality on smartphones and tablets
- Touch-friendly buttons and inputs
- Readable on all screen sizes
- Offline accessibility

### Printing
- Each calculator is print-friendly
- Include the full results and clinical interpretation
- Use browser print function (Ctrl+P or Cmd+P)
