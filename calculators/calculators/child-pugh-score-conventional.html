Child-Pugh Score (Conventional Units)

Assess liver disease severity
Child-Pugh score evaluates severity of liver disease. Uses conventional units (mg/dL for bilirubin, g/dL for albumin).
Child-Pugh Score Results
-
Classification:
- Score 5-6: Class A (well-compensated cirrhosis)
- Score 7-9: Class B (decompensated cirrhosis)
- Score 10-15: Class C (advanced cirrhosis)

1-Year Survival:
- Class A: 100%
- Class B: 80%
- Class C: 45%
