Child-Pugh Components
Child-Pugh Score = Sum of component points (1-3 each) Bilirubin: 1 if <1.4, 2 if 1.4-2.8, 3 if >2.8 mg/dL Albumin: 3 if >3.5, 2 if 2.8-3.5, 1 if <2.8 g/dL INR: 1 if <1.7, 2 if 1.7-2.3, 3 if >2.3 Ascites: 1 (None) to 3 (Severe) Encephalopathy: 1 (None) to 3 (Grade 3-4) Class A (5-6 points): 1-year survival 100%, 2-year 85% Class B (7-9 points): 1-year survival 81%, 2-year 60% Class C (10-15 points): 1-year survival 45%, 2-year 35%
