PSI Formula

PSI Score = Age + Demographic/Comorbidity Points + Clinical Exam Points + Lab Points

Risk Classes:

  • Class I: Score <51 - ~0.1% mortality
  • Class II: Score 51-70 - ~0.6% mortality
  • Class III: Score 71-90 - ~2.8% mortality
  • Class IV: Score 91-130 - ~8.2% mortality
  • Class V: Score >130 - ~29.2% mortality
