# MiCE Project - GSK Pilot

## Project Overview

**MiCE** (Moment Marketing Intelligent Content Engine) is CLIRNET's AI-assisted structured content generation platform. This folder contains documentation for the GSK (GlaxoSmithKline) pilot project.

## MiCE Platform Capabilities

- Topic discovery and trend identification
- AI-assisted content generation
- Compliance validation workflows
- Medical/scientific content review
- Approval workflow management
- Multi-format content output

## Critical MSA Sections

### Cyber Security Schedule (Pages 18-21)
- Section 3.1: Security program standards (ISO 27001, NIST, COBIT)
- Section 7.2: Patching timelines (30 days critical/high)
- Section 8.4: Encryption requirements
- Section 8.6: 24-hour breach notification

### AI Schedule (Pages 30-33)
- Section 1.1: AI usage requires written GSK approval
- Section 1.2: GSK can revoke AI authorization without liability
- Section 2.2: Cannot use GSK data to train/improve MiCE for other clients
- Section 3.2-3.3: Training data compliance warranties
- Section 5.5: 48-hour AI incident reporting
